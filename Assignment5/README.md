# Site Plan

-  My site plan is to basically just act as a blog of sorts for my own personal use that I can jot down different resources and ideas, as well as WIP photos of different artwork and what my thought process is behind them!

## BLOG PAGE

Goal: Act as an introduction to the website and be a sort of landing page for the different artworks and their respective page links.

### Structure

- **Navigation:** Links to different parts of the website will be located at the top left. I don't have a logo at this exact moment in time so I am leaving it the compass.  
- **Hero:** Hero is going to be the FEATURED CARD section. This is technically a part of the main content, but the rest of it will go under this so this stands out.
- **Main Content:** The main content is going to be different links to different WIPS or completed work. Each will have a their own little card with a READ MORE attached to it that links to their individual page.
- **Call-to-Action:** At the top with the title card. Basically just links to get into contact with me or see my stuff.
- **Footer:** Auto Updating Copyright in the bottom center. I think that should be fine for now.

# What I Learned

I think one of the main things I learned about web design while doing this site is just how good classes are. At the beginning of the semester, I understood what they did, but I struggled to do it. Now I realize how good they actually are, and how helpful they are too when it comes to setting everything up! I love how you can essentially just create a single class and then copy/paste it into different sections to make everything match. Kind of like how my "About the Artist" section is exactly the same class as my Contact pages, and so there is that inherent cohesion with it. Same with working with fonts and colors... classes make it so much easier!

# Balance

My main thing with this site was being able to have lots of artwork/images while still giving the user enough legibility with the text. I think using the Dark theme and colors helps with digital legibility, especially over images, but I think the use of cards helps this as well. On each project page, I have a lot of images, but the different transparencies and card "blocks" help break it up in a nice way. Obviously if this was a genuine website I was actively working on at all times, I'd love to deep dive into sketches and more nerdy stuff, but for this project I think it gets the point across! I'm glad I was able to add everything I wanted (for the most part) while keeping it easy to navigate!

# What am I Proud of?

I would say I'm most proud with the general cohesion of the site. I like being able to see all of my artwork together in one place, but it's just a nice setup to have. I mentioned it before, but I kind of made it similar to the way that Steam allows developers to post updates on their games. Even just the Steam pages for games have a similar setup, and I really like it. It's informational, but the breaks in text with the images just add a lot of flavor and I am happy that I was able to replicate that vibe somewhat!

# Future Use?

I learned a lot and I can definitely see myself branching off and actually creating this website, either for just myself or to share with fans of my work. I think I was able to create things that I am proud of, but I know I am just scratching the surface with this. It's a little confusing, and I'll admit that my style sheet could be organized way more, but I can definitely see myself doing something with this in the future. Sometimes school can be frusturating because there are things that you genuinely will not use in your life at all, but this class really helped me have a general understanding of web design and I can see myself using this knowledge in the future!
